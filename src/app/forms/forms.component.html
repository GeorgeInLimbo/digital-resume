Reactive Forms Example:

This is an example of a Reactive Form using Angular. The benefit of using Reactive Forms is the scalability resulting from using observable streams and sychronous updates based on change detection through the ReactiveFormsModule that each change to the form returns a new state to maintain of the model. Since Reactive Forms are built around observables, where form inputs and form values are provided as streams, which can be accessed synchronously. You can view the real time changes in the browser with the JSON values reflected here:

JSON Values from Reactive Forms: {{ myForm.value | json }}
